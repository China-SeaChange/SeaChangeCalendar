# 📅 SeaChange Calendar 操作手册

> 一份纯前端、无服务器的个人日程管理工具，支持拖拽、批量操作、撤销/重做、表情标签等强大功能。所有数据仅保存在你的浏览器中。

## 🖥️ 界面速览

- **顶部工具栏**：月份切换、年份/月份快速跳转、搜索、日程列表、导入/导出、清空数据、折叠顶部栏。
- **日历主体**：以月视图展示，每个日期格内显示：
  - 日期数字 + 表情标签（如有）
  - 内容条（可自定义颜色、里程碑标识）
  - 鼠标悬停显示备注（如有）
- **右键/双击**：双击日期格打开添加窗口；双击内容条快速编辑。

## ✨ 基本操作

### 添加日程
1. **双击任意日期格**（仅当月），弹出编辑窗口。
2. 输入**内容文字**（必填），可添加**备注**（悬停显示）。
3. 选择颜色、勾选“里程碑”（🚩标识）。
4. 点击 **「➕添加」**，日程即出现在该日期格内。

> 💡 同时也可以**拖拽选中多个连续日期**（详见“批量操作”），然后松手弹出窗口，一次性为多天添加相同内容。

### 编辑/删除日程
- **双击内容条**：直接打开编辑窗口，修改文字、颜色、备注或里程碑状态，点击 **「更新」** 保存。
- **单击内容条**：选中该条（橙色外框），然后：
  - 按 `Delete` 或 `Backspace` 删除。
  - 按 `Ctrl+C` 复制该条（包含颜色、里程碑、备注）。
  - 选中另一个日期格（单击或拖拽多选）后按 `Ctrl+V` 粘贴。
- **在编辑窗口中删除**：打开已有日程的编辑窗口，在“已有内容条”区域点击垃圾桶图标。

### 移动与排序
- **移动到其他日期**：直接拖拽内容条到目标日期格（支持跨月）。
- **同一天内调整顺序**：拖拽内容条到另一条的上方或下方（出现拖拽指示线时释放）。

> ⚠️ 拖拽时请按住鼠标左键，拖到目标位置释放即可。

## 🏷️ 日期表情（Emoji）

- **为日期添加表情**：双击日期格打开窗口 → 在“日期表情”区域点击任意预设 emoji，或输入自定义表情（最多4个字符）后点击「添加」。
- **移除表情**：点击已添加的表情图标即可删除。
- **清空当日所有表情**：点击「🗑️ 清空全部表情」按钮。
- 表情会显示在日期数字旁边，可用于标记节日、生日等。

## 🖱️ 批量操作（多选日期）

### 拖拽连续选择
1. 在日历上**按住鼠标左键**（从某个日期格开始）。
2. 拖动鼠标到另一个日期格，中间所有连续日期会被高亮（浅蓝色）。
3. 释放鼠标 → 弹出“跨日长条”窗口，可**一次性为所有选中的日期添加相同内容**。
4. 也可以先选中多天，然后按 `Ctrl+V` 将剪贴板中的内容粘贴到这些天中。

### 多选不连续日期（配合快捷键）
- 按住 `Ctrl` 键（Windows）或 `Command` 键（Mac），**依次点击日期格**，可以选中多个不连续的日期（边框变为蓝色）。
- 选中后，可以：
  - 按 `Delete` 删除所有选中日期内的**所有内容条和表情**（需二次确认）。
  - 按 `Ctrl+C` 复制这些天内所有内容条（跨天批量复制）。
  - 按 `Ctrl+V` 粘贴到其他选中的日期（会合并添加，不覆盖）。

## 🔍 搜索日程

1. 在顶部搜索框输入关键词（支持内容文字或备注）。
2. 点击搜索图标或按 `Enter`。
3. 弹出结果列表，点击任意结果直接跳转到对应日期并高亮该日期。

## 📁 数据导入/导出

- **导出**：点击「导出」按钮，下载一个 `.json` 文件，包含所有日程和表情数据。
- **导入**：点击「导入」按钮，选择之前导出的 `.json` 文件。导入时会**智能合并**：相同内容（文字、颜色、里程碑、备注完全一致）不会重复添加，仅添加新内容。
- **清空所有数据**：点击「清空」按钮，需二次确认，将删除全部日程和表情（不可撤销，请先导出备份）。

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `←` / `→` | 切换上/下个月（输入框未聚焦时） |
| `空格` | 跳转到今天 |
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + Y` 或 `Ctrl/Cmd + Shift + Z` | 重做 |
| `Ctrl/Cmd + C` | 复制选中的内容条或所选日期中的所有内容条 |
| `Ctrl/Cmd + V` | 粘贴到选中的日期格（单个或多个） |
| `Delete` / `Backspace` | 删除选中的内容条；如果无选中内容条但选中了日期，则删除该日期全部内容及表情（弹窗确认） |
| `Esc` | 关闭任何打开的弹窗（编辑、搜索、所有日程、确认框） |

## 🔧 小技巧

- **快速定位到某年某月**：直接点击顶部的**年份数字**或**月份数字**，输入数字后按 `Enter` 即可跳转。
- **折叠顶部栏**：点击「折叠」按钮，顶部工具栏会收起，需要时点击右上角「▼」按钮重新展开。
- **查看所有日程**：点击「日程」按钮，按日期排序展示全部内容条，点击任意一条可跳转。
- **鼠标悬停备注**：为内容条添加备注后，鼠标悬停在该条上会显示备注文字。
- **音效反馈**：每次操作有轻微音效（需要先与页面交互一次，如点击任意按钮，浏览器才会允许播放声音）。

## ❓ 常见问题

**Q：数据会丢失吗？**  
A：数据保存在浏览器的 `localStorage` 中，**清除浏览器缓存或更换设备会导致数据丢失**。建议定期导出备份。

**Q：可以修改年份上限吗？**  
A：本工具支持任意年份（1～9999），通过点击年份数字直接输入即可。

**Q：移动端能用吗？**  
A：支持触摸屏操作。拖拽选择日期需按住并滑动，拖拽内容条时请长按后移动。部分功能（如右键菜单）可能受限于移动端特性。

**Q：为什么没有声音？**  
A：首次操作时需点击页面任意按钮（如翻月），浏览器才会允许播放音频。另外，部分浏览器可能限制自动播放策略。
